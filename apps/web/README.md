# callcheck web

Landing page and docs for callcheck, the tool call reliability layer for AI agents.

Built with Next.js, Tailwind, and shadcn/ui.

## Develop

```
npm install
npm run dev
```

Runs at localhost:3000.

## Pages

`/` — landing page, install command, feature overview
`/docs` — full usage docs, schema format, repair, CLI, wrap once, report

## Deploy

Deployed on Vercel. Push to main, Vercel builds and deploys automatically.
